## Asteroids

A simple Asteroids clone written in JavaScript with HTML5 canvas. Shoot all the floating red balls to win.


### Keys

- `p` -- pause
- `left arrow` -- rotate left
- `right arrow` -- rotate right
- `up arrow` -- go forward
- `space` -- shoot
